## Overview

**Kograph Tools** is an all-in-one utility website designed with a clean multi-page interface and a mobile-friendly experience.
It is built for users who want fast tools without clutter, unnecessary explanations, or distracting UI.

The project focuses on:

- clean interface
- practical tools
- responsive layout
- polished user flow
- simple deployment workflow

---

## Features

### Core Tools

- **QR Generator**
  - Generate QR from links or text
  - Custom colors
  - Custom dot and corner styles
  - Logo in the center
  - Real-time preview
  - Download as PNG

- **Barcode Generator**
  - Generate common barcode formats
  - Instant preview
  - Color customization
  - PNG export

- **Video Tools**
  - Video compression
  - HD upscale workflow
  - Better process feedback
  - User-friendly progress messages

- **Remove Background**
  - Standard mode
  - HD mode
  - Cleaner preview flow
  - Download transparent result

- **Document Converter**
  - DOCX to text/html workflows
  - Image to PDF
  - Merge PDF
  - General conversion-ready structure for expansion

### UI Highlights

- Multi-page layout
- Clean home page
- Better mobile navigation
- Modern cards and spacing
- Improved footer structure
- Terms & Privacy pages
- Support button for creator

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Fizxxc/Kograph-Toolss.git
cd Kograph-Toolss
```

### 2. Install frontend dependencies

```bash
npm install
```

### 3. Create and activate Python virtual environment

#### Windows

```bash
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

#### macOS / Linux

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 4. Run development server

```bash
npm run dev
```
